Welcome to SW5E Companion

SW5E Companion is a character creator allows for the creation of characters that follows the basic rule-set within the Player's Handbook. There is no extended content or other fan-made content for SE5E. It is a light-weight tool that allows for the creation of a basic character composition

Features

  • Species, Class, and Background Selection
  • Skill selection associated with class and background
  • Force or Tech power selection if applicable to class
  • Ability Score Selection based on the Standard Array
  • Equipment Selection
  • Overview of created Character
  • User Dashboard to display created characters
